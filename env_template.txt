GEMINI_API_KEY=your_gemini_api_key_here
GEMINI_MODEL=gemini-pro

Instructions:
1. Copy this file and rename it to .env
2. Replace 'your_gemini_api_key_here' with your actual Gemini API key
3. Get your API key from: https://makersuite.google.com/app/apikey
4. Optionally change GEMINI_MODEL to use a different model:
   - gemini-pro (default, most widely available)
   - gemini-1.5-pro
   - gemini-1.5-flash
   - gemini-2.0-flash-exp
   - gemini-2.5-pro (if available)
   - Leave blank or use 'gemini-pro' if unsure
